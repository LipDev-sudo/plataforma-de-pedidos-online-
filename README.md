# Plataforma de Pedidos Online

Sistema de pedidos online para restaurantes e pequenos comercios, com cardapio digital e gestao de pedidos.

## Preview

[Ver Demo](https://plataforma-de-pedidos-online-two.vercel.app/)

## Tecnologias

- React
- TypeScript
- HTML / CSS
- JavaScript
- Vite

## Como rodar

```bash
npm install
npm run dev
```

## Funcionalidades

- Cardapio digital interativo
- Sistema de pedidos em tempo real
- Carrinho de compras
- Interface responsiva para mobile e desktop
- Layout otimizado para restaurantes e delivery
